Data Cleaning

The 10-Step Manual Guide (< 1M Rows)

1.Save & Setup: Save Copy → Select All (Ctrl+A) → Auto-fit → Table (Ctrl+T).
2.Clean Text: Remove Parentheses via Find & Replace (*).
3.Standardize Case: =LOWER(C2) to fix capitalization.
4.Clean Names: =PROPER(TRIM(D2)) for names.
5.Split Columns: Data Tab → Text to Columns (Delimited).
6.Remove Duplicates: Data Tab → Remove Duplicates.
7.Fill Blanks: Ctrl+G (Go To) → Special → Blanks → Type "NA" → Ctrl+Enter.
8.Handle Errors: =IFERROR(VLOOKUP(...), "Not Found").
9.Format Headers: Bold, Dark Blue Background, White Text.
10.Final Polish: View Tab → Uncheck Gridlines.

🚀 Advanced Excel Cleaning

Date Repair

Fix text dates like "2023.12.01"

=DATEVALUE(SUBSTITUTE(A2,".","/"))
Master Lookup

Standardize categories from a master list.

=XLOOKUP(A2, MasterList[Bad], MasterList[Good])
Advanced Text

Extract text before a symbol.

=TEXTBEFORE(A2, "@")
Data Validation

Prevent bad data entry.

Data Tab → Data Validation → Allow: List

Large Data Strategy (> 1M Rows)

1. Essentials

Remove Duplicates (Safe)
CREATE TABLE clean AS
SELECT DISTINCT * FROM original;
Handle Nulls
UPDATE clean
SET col = 'Unknown' WHERE col IS NULL;
Standardize Case
UPDATE clean
SET col = UPPER(TRIM(col));

2. Advanced Transformations

Fix Data Types
ALTER TABLE clean
MODIFY COLUMN price DECIMAL(10,2);
Conditional Logic
SELECT name,
CASE WHEN age < 18 THEN 'Minor'
ELSE 'Adult' END as status
FROM users;
String Replacement
UPDATE clean
SET phone = REPLACE(phone, '-', '');

Quality Assurance Checks

Invalid Emails:
WHERE email NOT LIKE '%@%'
Negative Prices:
WHERE price < 0
Future Dates:
WHERE order_date > CURRENT_DATE

The Vibe Coder Project Cycle (The Ultimate Practice Routine)

Do not just "practice." Build a legacy. Follow this 6-step cycle for every project.

Step 1: SCOUT (Find the Dataset)

Goal: Find data that mimics real-world business scenarios.

  • Beginner: Kaggle ("Sales", "Retail", "HR").
  • Intermediate: Maven Analytics Data Playground.
  • Advanced: Government Portals (data.gov) - Messy & Realistic.
AI Prompt: "I need a dataset to practice [Skill]. Suggest 3 specific dataset topics on Kaggle that are suitable for a Data Analyst portfolio."

Step 2: SCOPE (Define the Mission)

Goal: Don't dive in blind. Define the "North Star" (Rule of 3).

  • Descriptive: "What happened?" (e.g., Total Sales by Month).
  • Diagnostic: "Why did it happen?" (e.g., Which Region caused the sales drop?).
  • Prescriptive: "What should we do?" (e.g., Which products should we discount?).
AI Prompt: "I have a dataset about [Topic]. Act as a Stakeholder and give me 3 difficult business questions to solve."

Step 3: SCRUB (The Clean-Up)

Goal: Make the data analysis-ready. Document every change.

  • Excel: Remove Duplicates, Fix Formatting, Fill Blanks.
  • SQL: UPPER(), TRIM(), COALESCE().
  • Doc: Create changelog.txt.
AI Prompt: "I have a date column formatted as '01-Jan-2024'. Write a SQL query to convert this into standard 'YYYY-MM-DD' format."

Step 4: SOLVE (The Analysis)

Goal: Answer the questions from Step 2.

  • Excel: PivotTables, VLOOKUP.
  • SQL: GROUP BY, Window Functions.
  • Python: Pandas Feature Engineering.
AI Prompt: "Write a SQL query to rank the top 5 salespeople per region using a Window Function."

Step 5: STORY (The Visualization)

Goal: Build a dashboard that speaks for itself.

  • Requirement: KPI Cards, Interactivity (Slicers), Consistent Colors.
AI Prompt: "Suggest a color palette (Hex codes) for a financial dashboard that looks modern and professional."

Step 6: SHARE (The "Flex")

Goal: If you don't publish it, it didn't happen.

  • GitHub: Upload SQL/Python + README.md.
  • LinkedIn: Hook + Process + Visual + CTA.
AI Prompt: "Draft a LinkedIn post for this project. The key finding was that mobile users churn 20% faster than desktop users. Keep it engaging."
